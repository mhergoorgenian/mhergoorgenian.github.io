IMAGE · AI

Background Remover

High-quality AI background removal powered by BiRefNet via rembg. Drag-and-drop web UI with multiple models — handles hair, fur, glass, and complex edges. Supports alpha matting for sharper cutouts. CLI included.

  • BiRefNet General / Portrait / Lite
  • ISNet & U2Net model options
  • Alpha matting with fine-tuned thresholds
  • Transparent PNG export
  • Gradio web UI + Click CLI

PROCEDURAL · LEVEL DESIGN

Dungeon Map Generator

Procedural Metroidvania-style 2D side-view dungeon silhouettes. Generates horizontal floors, vertical shafts, chambers, and looping paths with configurable seeds and exportable room data.

  • Seed-based reproducible maps
  • Floor count, rooms, shafts, loops
  • Edge roughness & extra branches
  • PNG map + JSON room metadata export
  • CLI for batch generation

VIDEO · SPRITES

Video to Sprite Sheet

Turn gameplay or reference videos into transparent sprite sheets. Extracts frames, removes backgrounds with BiRefNet, trims padding, and packs into a grid with JSON frame metadata for game engines.

  • Configurable frame count & grid layout
  • HQ upscaling for small source frames
  • Alpha matting & output scale control
  • 16-frame default (4×4 grid)
  • JSON metadata per frame bounds
